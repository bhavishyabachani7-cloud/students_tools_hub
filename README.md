# 🚀 Student Toolkit

An all-in-one productivity toolkit designed for students to save time and get more done — fast, simple, and free.

---

## 🌟 Features

- 📄 Resume Builder (Download professional PDF resumes)
- ✍️ Smart Text Summarizer
- 🔊 Text to Speech (with audio download)
- 🖼️ Image Compressor (powered by TinyPNG API)
- 📑 PDF Merge Tool
- ✂️ PDF Split Tool

---

## 🛠️ Tech Stack

- Python 3.11
- Flask
- HTML, CSS
- ReportLab (PDF generation)
- PyPDF2 (PDF tools)
- gTTS (Text-to-Speech)
- Tinify API (Image compression)
